Démarrage

Dézipper le dossier stoploss a l'endroit souhaiter. Pour chaque actualisation futur il suffira de dézipper par dessus votre ancien dossier stoploss pour garder les configurations telles quelles.

Lancer l'application (StopLoss.exe):

Si vous obtenez une erreur au lancement de l'appli, veuillez téléchargez "net framework 4.0" de microsoft.

Configuration de la DB

Dans les champs server, port, userdb, password vous devez renseigner les valeurs nécessaire pour que le soft puisse se connecter a votre DB postgresql. Les valeurs ici insérées correspondent aux valeur par défaut (voir les valeur en passant la souris sur le "i").

Le champ database correspond au nom de votre base de donnée de votre tracker.


Après avoir remplis tout les champs, cliquer sur test si c'est ok vous obtiendrez ce message et l'application se ferme.


En cas contraire veuillez vérifier les données insérez plus particulièrement le nom de la base de donnée.

Re-ouvrir l'application, et désormais vous pourrez saisir votre pseudo de la room où vous jouez ou votre alias de hem ou pt4 (si tout est ok vous aurez ceci).


Conf. Stop

Dans l'onglet conf. stop vous allez pouvoir configurer vos stop. Ici vous pouvez tous les configurer ou aucun, a votre choix.


Stoploss:

A partir de xx BBs négatif le son retentit.

StoplossPeak:

On commence notre session et au bout d'un certain on gagne 200bbs. Donc si dans over on n'a mis 160bbs, le stoplosspeak commence a rentrer en compte vu que 200 > 160. Hors maintenant si je perd 180 bbs et j'ai inséré 150bbs dans stoplosspeak, des 200 gagné je tombe a 20bbs. Dans ce cas la le son retentit car j'ai perdu plus que les 150bbs défini dans stoplosspeak.

StopHands:

A partir de xx hands le son retentit.

StopTime:

A partir de xx minutes le son retentit (insérer en minute).

StopWin:

A partir de xx BBs positif le son retentit.

Resume on Stop:

Si on coche, a la fin de la session, en cliquant sur stop on se verra amener vers l'onglet resume session où il y aura le résumé de la session.

Hide BBs:

Permet de cacher les bbs dans la fenêtre qui apparait apres avoir cliquer su start.

Close Skype:

En cochant, l'application ferme pour vous skype.

Active button set:

Permet de activer le bouton "set" pour réajuster les valeurs des stop en cours de session.

Sounds:

Permet de définir les son souhaitez. (seulement des sons *.wav).

Resume Session

Cet onglet vous permettra avoir un résumé de votre session et de vos meilleur résultat positif et négatif depuis que vous avez stoploss.


Start

Start pour commencer la session.
Si vous aimez le soft un petit don merci.
En bas vous avez aussi la possibilité d'envoyer un mail pour une critique ou suggestion.


Durant la session


Si vous avez décocher la case "hide session" en passant votre souris ou il y a le 0 comme dans l'image, cela affichera les BBs gagné ou perdu pendant la session actuel.

Hands:

Nombre de mains jouer pendant la session.

Temps:

Le temps écoulé et si vous cliquer sur le temps, le temps s'arrête. Exemple: quand vous êtes sitout.

Set:

Si activé, permet de réajuster les valeurs de stop.

Stop:

Stoppez la session.
